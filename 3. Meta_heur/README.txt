* Problema de las N-Reinas

Este algoritmo se encuentra en el archivo "N_Reinas.py".


Para compilar y ejecutar:

	python3 N_Reinas.py archivo.txt

En la parte de "archivo.txt" se coloca el nombre del archivo a ejecutar de los contenidos en la carpeta "Ejemplares".

Ejemplo: 

	python3 N_Reinas.py N_Reinas_archivo.txt
